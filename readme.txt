# Accounting App

A cross-platform accounting app built with Ionic and Angular using the MVVM architecture. It features:

- Sales invoices
- Item management
- Error handling
- Authentication
- Role-based permissions
- Robust routing

The app is integrated with an API and includes additional functionalities:

- Dedicated backup offices
- Secure data upload to Google Drive using Java Native techniques, ensuring data protection and easy retrieval.
